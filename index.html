Deep Dexterous Grasping

Grasping objects is difficult. Dexterous grasping using a single-view depth camera is harder. On this page, you'll find all you need about our new paper "Deep Dexterous Grasping", accompanying grasping simulator, 2M simulated grasp dataset, re-running grasps in simulation and collecting new data.

The paper includes two new methods that improve the state-of-the-art in dexterous grasping of novel objects using a single view depth camera.

The paper has been publicly available via this link.

Code & Data

The grasping simulator is a carefully tuned dexterous grasping environment based on the DLR-II hand. It uses MuJoCo engine for physics simulation. You can re-run simulated grasps and record videos using this tool. The ability to generate novel grasps on new scenes is coming soon. Click here for the code and dataset installation instructions.
